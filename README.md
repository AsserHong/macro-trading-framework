# 宏观交易框架学习项目

## 核心思想

所有大类资产最终都锚定于美元。理解美元 = 理解市场。

美元分解为两大驱动维度：

- **外汇（汇率维度）** — 美元相对其他主要货币的强弱，盯 EUR/USD
- **利率（资金成本维度）** — 美债收益率曲线（2Y / 5Y / 10Y / 30Y）

补充第三因子：**风险偏好** — 信用利差、VIX，捕捉极端市场场景。

## 两个层次

本项目分为两层，缺一不可：

**第一层 · 宏观认知（大势）** — 你站在哪个市场环境里
- 主线四类资产：股票 · 债券 · 外汇 · 大宗商品
- 扩展：数字货币与美元的博弈
- A股市场：政策周期 + 流动性双轮驱动框架

**第二层 · 交易能力（吃饭）** — 你如何在市场里活下来并做大
- 交易系统：可复制的进出场与仓位规则
- 风险与资金管理：活下来才能做大（决定能否走到游资那天）
- A股游资战法：情绪周期、龙头、打板接力、龙虎榜（A股短线真正的核心）
- 交易心理 + 复盘系统：对抗人性、持续进化

> ⚠️ 重要认知：**A股游资 ≠ 宏观交易员**。宏观（01-05）告诉你"大势能不能上车"，
> 游资战法（06-10）才是短线吃饭的本事。两层都要，但别把宏观当成短线的全部。

## 学习路线

**按序通读**（建立完整框架）：
```
01 → 02 → 03 → [04] → 05 → 06 → 07 → 08 → 09 → 10
```

**从零起步的散户**（先保命，再求胜）——推荐优先顺序：
```
07 风险与资金管理 → 09 交易心理 → 06 交易系统 → 10 复盘系统
→ 01-05 宏观认知 → 08 A股游资战法 → appendix/资金进阶阶梯
```
理由：新手最先该学的不是怎么赚钱，而是**怎么不爆仓 + 怎么管住手**；
游资战法（08）放在有了系统和纪律之后再学，否则只会变成赌博。

## 时间建议

每周 2-3 小时碎片化学习。宏观主线约 10-12 周；
交易能力层（06-10）需配合**实盘/模拟盘同步练习**，按月迭代，不设结业期限。
