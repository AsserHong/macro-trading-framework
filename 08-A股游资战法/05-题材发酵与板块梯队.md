# 题材发酵与板块梯队

> 一个题材从无人问津到全市场疯抢，中间经历了什么？为什么有的题材一日游，有的能炒一个月？看懂"发酵节奏"和"梯队结构"，你才知道现在是上车还是下车。

---

## 本课将讲清

- 题材的生命周期：消息 → 卡位 → 扩散 → 分歧 → 退潮
- 板块梯队：龙头 / 中军 / 跟风 / 卡位，资金如何在梯队间轮动
- 真题材 vs 伪题材：持续性取决于**想象空间 + 业绩兑现预期**
- "高度决定情绪"——龙头的连板高度如何带动整个板块节奏
- 如何用梯队结构判断题材是在加速还是在熄火
