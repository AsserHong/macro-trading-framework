# 宏观交易框架学习项目

## 核心思想

所有大类资产最终都锚定于美元。理解美元 = 理解市场。

美元分解为两大驱动维度：

- **外汇（汇率维度）** — 美元相对其他主要货币的强弱，盯 EUR/USD
- **利率（资金成本维度）** — 美债收益率曲线（2Y / 5Y / 10Y / 30Y）

补充第三因子：**风险偏好** — 信用利差、VIX，捕捉极端市场场景。

## 资产覆盖

主线四类：股票 · 债券 · 外汇 · 大宗商品

扩展：数字货币与美元的博弈关系

新增：A股市场 — 政策周期 + 流动性双轮驱动框架

## 学习路线

```
01 美元基础 → 02 资产传导 → 03 综合实战 → [04 数字货币扩展] → [05 A股市场]
```

## 时间建议

每周 2-3 小时碎片化学习，预计 10-12 周完成主线内容。

## 使用方式

1. 按序阅读，每篇文章独立可读
2. 每篇以交易员真实问题开场，围绕问题推导
3. 在 PROGRESS.md 中标记学习进度
4. 附录提供指标速查和术语表供日常参考
