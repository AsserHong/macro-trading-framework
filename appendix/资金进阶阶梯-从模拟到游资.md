# 资金进阶阶梯：从模拟到游资

> 从 0 到游资不是一步到位，而是一级一级爬上去的。每一级的目标、资金量、允许用的手法、要过的"考核"都不一样。这张阶梯，是你的整体成长路线图。

---

## 本篇将讲清

分阶段的成长路径，每阶段明确：**目标、资金规模、允许手法、风控约束、晋级条件**。

| 阶段 | 核心目标 | 关键约束 |
|------|----------|----------|
| 0. 模拟盘 | 验证系统、养成纪律 | 不投真金，只练流程与复盘 |
| 1. 小资金试错 | 用"学费可承受"的钱体验真实情绪 | 严格止损，单笔风险极小 |
| 2. 稳定盈利 | 跑出正期望、稳定的资金曲线 | 月度复盘验证系统有效性 |
| 3. 逐步加码 | 在验证有效后放大仓位 | 回撤触发降仓机制 |
| 4. 规模化 | 向游资体量过渡 | 流动性、冲击成本、合规边界 |

*（以上为骨架，各阶段细节待填充）*

## 核心原则

- **先证明能稳定赚钱，再放大资金**——而不是用加仓掩盖系统缺陷
- 每一级的"晋级"必须有数据支撑（胜率、盈亏比、最大回撤），而非感觉
- 资金越大，可用手法越受限（见 [合规红线](合规红线-游资不能碰的边界.md)）
